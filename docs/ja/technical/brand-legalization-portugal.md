## 2. 法的構造 — 何を開設するか?

### オプション 1: 個人事業主（Recibos Verdes）← 開始時の推奨

*「Recibos Verdes」（直訳「緑の領収書」）はポルトガルの個人事業主・フリーランス税制度である。*

- ✅ よりシンプルで安価
- ✅ 初期費用なし
- ✅ 利益に対し IRS（Imposto sobre o Rendimento das Pessoas Singulares — ポルトガル個人所得税）を支払う（基本税率 23%）
- ✅ 社会保障: 収入の約 21.4%
- ❌ 無限責任（個人資産も負債の対象となる）

| | |
|---|---|
| **開設費用** | 0 € |
| **所要時間** | 1日（オンライン） |
| **推奨 CAE コード** | 25620（機械加工） |

*CAE = Classificação Portuguesa das Atividades Económicas、ポルトガルの経済活動分類コード。*

### オプション 2: 一人有限責任会社（Unipessoal por Quotas）← 事業が拡大した場合

- ✅ 個人資産と事業資産を分離できる
- ✅ より専門的
- ❌ 費用: 設立に約 360 € + 月額の会計士費用（50〜150 €/月）

## 3. 法的登録の手順（個人事業主ルート）

### ステップ 1: 税務当局（Finanças）への活動登録

- **税務当局ポータル:** <https://www.portaldasfinancas.gov.pt>
- Declaração de Início de Atividade（活動開始申告）を提出する
- **CAE コード:** 25620（機械加工）または 74100（専門デザイン）
- **費用:** 0 €

### ステップ 2: 社会保障への登録

- Finanças での登録後、自動的に行われる
- 月額約 70〜100 €（収入による）
- **費用:** 0 €（後の月額拠出のみ支払う）

### ステップ 3: 請求書発行

- 請求書および領収書を発行する必要がある
- **無料ソフトウェア:** InvoiceXpress（一定限度まで無料）または Moloni
- 紙の請求書も可能（推奨されない）

### ステップ 4: IVA（付加価値税）

*IVA = Imposto sobre o Valor Acrescentado、ポルトガルの付加価値税（VAT）。*

- 年間請求額が **13,500 € 未満** の場合: IVA 免税（簡易制度）
- 年間請求額が **13,500 € 以上** の場合: 23% の IVA を請求し、国に納付する
